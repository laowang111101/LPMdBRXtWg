# 内蒙古旅游景点数据分析系统 python695

## 项目概述

内蒙古旅游景点数据分析系统是一款基于Python语言的综合性数据分析平台。该系统结合了Django框架、Hadoop大数据处理以及Scrapy爬虫技术，对内蒙古自治区的旅游景点进行深入的数据挖掘与分析。

## 技术栈

- **后端：** Python Django
- **大数据处理：** Hadoop
- **数据采集：** Scrapy
- **数据库：** MySQL

## 功能模块

### 系统首页

- 展示内蒙古旅游资源的统计与趋势分析。

### 用户模块

- 用户注册、登录、注销等功能。

### 景点模块

- 展示旅游景点详细信息，包括文字描述、图片、地图位置等。

### 周边住宿

- 提供景点周边的住宿信息。

### 周边美食

- 展示景点周边的美食推荐。

### 美食分类

- 对美食进行分类，便于用户查找。

### 系统管理

- 系统设置、权限管理、内容管理等。

### 个人资料

- 用户可查看与编辑个人资料。

## 系统角色

- **游客：** 浏览系统信息，进行注册。
- **注册用户：** 拥有查看更多功能及编辑个人资料的权限。
- **管理员：** 对系统进行管理、维护。

## 运行环境

- Python 3.x
- Django 2.x
- Hadoop 3.x
- MySQL 5.x

## 部署步骤

1. 环境搭建：安装Python、Django、Hadoop、MySQL等必要环境。
2. 数据库配置：根据需求配置MySQL数据库。
3. 爬虫部署：利用Scrapy进行数据采集。
4. 系统部署：运行Django项目，进行前端与后端的部署。
